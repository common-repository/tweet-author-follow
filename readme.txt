This plugin helps you to show the author twitter profile in your posts using the shortcode [twitter-author]
== Description ==

With this plugin you can use the shortcode  [twitter-author] to add a nice preview of the author's twitter profile in your post.
The only requirement is that the author has indicated his twitter nickname in the user management page of WordPress.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `twitter-Author-follow.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= Where can I use the shortcode? =

You should use the shortcode in the posts 
= where can I insert the author's Twitter profile Name? =

go to user management, select one of your Users and compile the "Twitter" field with the name that he uses on Twitter, for example: if he uses "@thisIsMyName" you should insert "thisIsMyName"
